    v0.6.12 - 2025-10-16¶

    Breaking!¶

    This release changes shell and error handling syntax, expect breaks.

    // OLD: Critical command
$!`make build`

// NEW: Same behavior (critical by default)
$`make build`

// OLD: Unsafe command
unsafe $`command_that_might_fail`

// NEW: Use catch block
$`command_that_might_fail` catch:
    pass

// OLD: Recover block
$`curl {url}`
recover:
    print_err("Failed but continuing")

// NEW: Use catch block
$`curl {url}` catch:
    print_err("Failed but continuing")

// OLD: Fail block
$`curl {url}`
fail:
    print_err("Failed, exiting")
    exit(1)

// NEW: Use catch block with exit
$`curl {url}` catch:
    print_err("Failed, exiting")
    exit(1)

    • feat: implement new error handling syntax (catch suffix) (5c0253c)
    • feat: implement new shell syntax + catch suffix syntax (7ef3f46)
    • feat: update syntax dump tests for new shell & error handling syntax (78a4857)
    • feat: update textmate grammar for new shell + error handling syntax (981fdfb)
