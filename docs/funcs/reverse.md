# reverse

Reverses a string or list. Preserves color attributes for strings.

## Signature

`reverse(_val: str|list) -> str|list`

## Examples

```rad
reverse("hello")           // -> "olleh"
reverse([1, 2, 3, 4])      // -> [4, 3, 2, 1]
reverse("racecar")         // -> "racecar"
```

## Category

strings
