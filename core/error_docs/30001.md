# RAD30001: Type Mismatch

A value of one type was used where a different type was expected.

## Example

```rad
fn double(n int) -> int:
    return n * 2

x = "42"
double(x)  # Error: expected int, got string
```

## How to Fix

Convert the value to the expected type, or fix the type annotation:

```rad
double(int(x))  # Convert string to int first
```

## Type Conversion Functions

| Function | Description |
|----------|-------------|
| `int(value)` | Convert to integer |
| `float(value)` | Convert to float |
| `str(value)` | Convert to string |
| `bool(value)` | Convert to boolean |
