# RAD20008: Invalid Time Unit

The time unit string isn't recognized.

## Example

```rad
# Wrong
duration("5 weeks")   # 'weeks' not supported

# Correct
duration("5d")        # 5 days
duration("2h")        # 2 hours
duration("30m")       # 30 minutes
duration("10s")       # 10 seconds
```

## Valid Units

- `s`, `sec` - seconds
- `m`, `min` - minutes
- `h`, `hr` - hours
- `d`, `day` - days

Check the function's documentation for the full list of supported units.
