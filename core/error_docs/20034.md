# RAD20034: Switch No Match

The `switch` value didn't match any case, and there's no `default`.

## Example

```rad
status = "pending"
switch status:
    case "active":
        print("Active")
    case "inactive":
        print("Inactive")
// Error if status is something else like "pending"
```

## How to Fix

Add a `default` case:

```rad
status = "pending"
switch status:
    case "active":
        print("Active")
    case "inactive":
        print("Inactive")
    default:
        print("Unknown status: {status}")
```
