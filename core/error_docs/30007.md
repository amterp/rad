# RAD30007: Wrong Argument Count

A function was called with the wrong number of arguments.

## Common Causes

1. **Missing required arguments** - Not providing all required positional arguments
2. **Too many arguments** - Providing more arguments than the function accepts
3. **Confusing positional and named** - Passing a named argument positionally

## Examples

```rad
# Function expects 2 arguments, but got 1
fn add(a, b):
    return a + b

add(5)  # Error: missing required argument 'b'
```

```rad
# Function expects 1 argument, but got 3
fn double(x):
    return x * 2

double(1, 2, 3)  # Error: expected at most 1 arg, got 3
```

## How to Fix

1. **Check the function signature** - Look at how many arguments are expected
2. **Use named arguments for clarity**:
   ```rad
   process(input=value, output=dest)
   ```
3. **Check for optional arguments** - Some arguments may have defaults and
   don't need to be provided

## Finding the Signature

Use `rad docs functions <function_name>` to see the function's documentation
and expected arguments.
