# RAD20011: Parse JSON Failed

`parse_json()` couldn't parse the input as valid JSON.

## Example

```rad
# These fail
parse_json("{invalid}")         # Missing quotes around key
parse_json("{'key': 'value'}")  # Single quotes not valid
parse_json("{key: value}")      # Unquoted strings

# These work
parse_json('{"key": "value"}')
parse_json('[1, 2, 3]')
parse_json('null')
```

## Common Issues

- JSON requires **double quotes**, not single quotes
- Object keys must be **quoted strings**
- No **trailing commas** allowed
- No **comments** in JSON

## How to Fix

```rad
data = parse_json(text) catch:
    print_err("Invalid JSON: {data}")
    exit(1)
```
