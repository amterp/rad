# RAD40004: Return Outside Function

A `return` statement was used outside of a function definition.

The `return` statement is only valid inside functions - it specifies what value
the function should produce when called.

## Example

```rad
# Wrong: return at the top level
x = 10
return x  # Error: return outside function
```

```rad
# Correct: return inside a function
fn get_value():
    x = 10
    return x

result = get_value()  # result is 10
```

## How to Fix

1. **Use `exit()` to end the script**:
   ```rad
   if error:
       exit(1)  # Exit with error code
   ```

2. **Wrap code in a function** if you need return semantics:
   ```rad
   fn main():
       # ... your code ...
       return result

   output = main()
   ```

3. **Check your indentation** - Make sure the return is actually inside
   the function body, not accidentally de-indented.

## Note

At the top level of a script, use `exit()` to terminate execution.
`exit(0)` indicates success, `exit(1)` or any non-zero code indicates failure.
