# RAD10016: Missing Type

The parser expected a type annotation but didn't find one.

## Example

```rad
# Wrong
fn get_value() ->:
    return 42

# Correct
fn get_value() -> int:
    return 42
```

```rad
# Wrong
args:
    count = 5

# Correct
args:
    count int = 5
```

Valid types: `int`, `float`, `str`, `bool`, `any`, `void`, `null`, `list`, `map`.
