# RAD10002: Missing Colon

The parser expected a colon (`:`) but didn't find one.

## Example

```rad
# Wrong
if true
    print("hello")

# Correct
if true:
    print("hello")
```

Colons are required after block headers: `if`, `else`, `elif`, `for`, `while`, `fn`,
`switch`, `case`, `default`, `args`, `rad`, `request`, and `display`.
