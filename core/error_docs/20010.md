# RAD20010: User Input Error

An error occurred while reading user input with `input()`.

This usually happens when stdin isn't available - for example, when running a script in a non-interactive context like a cron job or piped command.

## How to Fix

Check if interactive input is available:

```rad
if has_stdin():
    data = read_stdin()
else:
    print("No input available")
```

Or handle the error:

```rad
name = input("Enter name: ") catch:
    name = "default"
```
