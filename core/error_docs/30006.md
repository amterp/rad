# RAD30006: Invalid Argument Type

A function received an argument of the wrong type.

## Example

```rad
# Wrong: len() expects a collection
len(42)

# Correct
len([1, 2, 3])  # 3
len("hello")    # 5
```

The error message tells you what type was expected. Convert with `int()`,
`str()`, `list()`, etc. if needed.
