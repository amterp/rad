# RAD10010: Retired

This error code is no longer in use. Tree-sitter's error recovery
produces ERROR nodes for the shape this code was designed to catch,
so the dispatch in error_messages.go now falls back to **RAD10001**
("Invalid syntax") or **RAD10009** ("Unexpected token") instead.

The number stays reserved per the tombstone rule - we never reuse
retired codes, so old logs that mention RAD10010 remain greppable.

See `rad docs RAD10001` or `rad docs RAD10009` for the
modern equivalents.
