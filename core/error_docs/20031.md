# RAD20031: Continue Outside Loop

`continue` can only be used inside a loop. It skips to the next iteration.

## Example

```rad
# Wrong
if x < 20:
    continue  # not inside a loop

# Correct
for i in range(10):
    if i % 2 == 0:
        continue  # skip even numbers
    print(i)
```
